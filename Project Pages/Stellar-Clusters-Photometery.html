Stellar Clusters Photometry

Overview

This project involved observational astronomy and data analysis using telescope collected images of stellar clusters. The experiment required following established laboratory procedures in a group of 4 for telescope operation and data acquisition, while the data processing, analysis, and interpretation were completed independently. The focus of my work was on writing analysis code, handling experimental uncertainty, and interpreting physical results from real observational data.

Data Acquisition

  • Observations were carried out using a remote telescope
  • Telescope operation and acquisition software were used according to the laboratory manual
  • Multiple image sets were collected across different wavelength filters
  • Calibration data was provided and applied as part of the standard workflow

    • Due to the complexity and safety requirements of telescope operation, these steps followed predefined procedures.

Data Processing and Analysis

This stage was completed independently and formed the core of my contribution.

  • Wrote Python code to process calibrated image data
  • Performed photometry to extract stellar flux values
  • Converted flux measurements to apparent magnitudes
  • Propagated measurement uncertainty through all calculations
  • Generated colour magnitude diagrams for stellar clusters
  • Estimated physical properties such as cluster distance and age

When analysing the M3 dataset, limitations in data quality were identified and evaluated, and the dataset was determined to be unreliable for accurate analysis.

Results and Interpretation

  • Successfully generated colour magnitude diagrams from observational data
  • Identified key stellar population features within usable datasets
  • Evaluated uncertainty sources and their impact on final results
  • Critically assessed when experimental data was insufficient for reliable conclusions

The project emphasised understanding both the capabilities and limitations of real scientific data.

Skills Demonstrated

  • Scientific Python programming
  • Data analysis and visualisation
  • Error propagation and uncertainty analysis
  • Interpreting real world experimental data
  • Working within procedural constraints while contributing original analysis
  • Technical reporting using LaTeX
